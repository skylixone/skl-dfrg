# PROJECT: the-defrag

## 🎯 Vision
A browser-based HTML "screensaver" that mimics (cosplays) a classic disk defragmentation tool. It's a visual simulator that emulates the behavior of block-based data organization with a nostalgic aesthetic.

## 🛠️ Tech Stack
- **Frontend:** Preact (Vite + TypeScript)
- **Engine:** `p5.js` (Canvas-based rendering)
- **Styling:** Tailwind CSS (aerospace-ui aesthetic: Geist Mono, 0px radius, hard borders, amber accents)
- **Deployment:** Vercel/GitHub Pages (Static)

## 📋 Core Features
- **Defrag Simulation:** Blocks moving, changing states (Reading, Writing, Defragmenting, Free Space).
- **Skinning Engine:** Support for different visual themes (Classic Windows, Modern, Dark Mode, etc.).
- **Interactive Controls:** Start/Stop/Pause, Speed control, Block density.
- **Nostalgic UI:** Retro-inspired dashboard elements (Percentage complete, elapsed time, etc.).

## 👥 Audience
Everyone - general users who enjoy nostalgic tech aesthetics and "oddly satisfying" visuals.

## 📈 Success Criteria
- Smooth, performant simulation (60fps).
- High-fidelity emulation of the classic defrag experience.
- Extensible skinning system.
